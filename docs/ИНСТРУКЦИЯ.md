# Фоторобот — конструктор. Краткая инструкция

Инструмент помощи оператору: составление ориентировочного портрета человека по
словесному описанию. Работает полностью **локально, без интернета**; данные
хранятся только на этом компьютере.

---

## 1. Как запустить

Три способа (любой на выбор):

1. **Автономное приложение (рекомендуется).** Открыть
   `dist\FaceComposite\FaceComposite.exe`. Ничего устанавливать не нужно. Папку
   `dist\FaceComposite\` можно целиком скопировать на другой компьютер с Windows.
2. **Через `Запустить.bat`** — открывает то же окно (нужен установленный Python
   с пакетом `pywebview`). Для разработки.
3. **В браузере** — открыть `index.html` двойным щелчком (Edge / Chrome). Самый
   простой запасной вариант. *Важно:* голосовой ввод в браузере недоступен —
   только в приложении (.exe).

---

## 2. Общий порядок работы

1. Вверху заполните **карточку дела**: название, оператор, тип (разыскиваемый /
   пропавший / иное).
2. Введите **описание внешности** — текстом или голосом.
3. Нажмите **«Собрать по описанию»** — система соберёт стартовый портрет.
4. **Доведите сходство** вручную (списки, ползунки, приметы) и/или через
   **«Подбор «выбери похожее»»**.
5. **Сохраните версию**, при необходимости **сохраните проект**.
6. **Экспортируйте «Ориентировку»** (лист для печати) или PNG/SVG.

---

## 3. Ввод описания

- **Текстом.** Опишите внешность в свободной форме, например:
  *«мужчина лет сорока, овальное лицо, тёмные короткие волосы, густые брови,
  прямой нос, карие глаза, короткая борода, шрам на щеке, очки»*.
  Нажмите **«Собрать по описанию»**. Внизу слева в блоке **«Распознанные
  признаки»** видно, что именно и по какому слову распозналось.
- **Голосом (только в приложении).** Нажмите **«🎤 Диктовать описание»**,
  продиктуйте описание, нажмите кнопку ещё раз — текст подставится и портрет
  соберётся автоматически. В первый раз Windows может спросить разрешение на
  микрофон — разрешите.
- **Примеры для демонстрации.** Чипы под кнопкой (Мужчина с бородой, Молодая
  женщина, Пожилой, Девушка, Ребёнок, Подозреваемый) собирают готовый портрет
  в один клик.

---

## 4. Ручная корректировка (панель справа)

- **Черты лица** — выпадающие списки: пол, возраст, тон кожи, форма лица,
  подбородок, уши, форма и цвет глаз, брови, нос, губы, причёска и цвет волос,
  растительность, очки.
- **Тонкая настройка** — ползунки: ширина/длина лица, ширина челюсти, высота
  лба, размер и расстановка глаз, размер/длина носа, размер/высота рта, высота
  бровей. Меняются плавно — удобно «дожимать» сходство.
- **Особые приметы** — флажки: шрамы, родинки, веснушки, морщины.
- **Сбросить черты** — вернуть лицо к стандартному виду.

---

## 5. Подбор «выбери похожее»

Способ уточнения, когда трудно описать лицо словами (как в EvoFIT/EFIT-V):

1. Нажмите **«Подбор «выбери похожее»»** — появится 9 вариаций лица.
2. Отметьте 1–2 **наиболее похожих** (клик по карточке).
3. Нажмите **«Новое поколение»** — система сблизится к выбранному, разброс
   сузится. Повторяйте, пока не приблизитесь к цели.
4. **«Ещё варианты»** — обновить варианты; ползунок **«Сила изменений»**
   регулирует разброс.
5. **«Применить выбранный»** — перенести вариант в рабочий портрет.

Пол, возраст, кожа, волосы и очки при подборе **не меняются** (их обычно
называют словами) — варьируется только геометрия лица.

---

## 6. Версии, отмена, проекты

- **Сохранить версию** — снимок текущего портрета; список версий слева, клик по
  версии — вернуться к ней.
- **Отменить / Повторить** (кнопки ↶ ↷ под портретом или **Ctrl+Z / Ctrl+Y**).
- **Сохранить проект** — сохранить карточку локально; **Проекты** — список
  сохранённых (открыть / удалить).
- **.json** — выгрузить проект файлом; **Загрузить** — открыть такой файл
  (удобно переносить между компьютерами).

---

## 7. Экспорт результата

- **Ориентировка** — печатный лист (шапка, портрет в рамке, данные дела,
  перечень признаков, словесное описание, дисклеймер) в PNG. Для печати и дел.
- **PNG** — только портрет (картинкой).
- **SVG** — портрет в векторе (для масштабирования без потери качества).

---

## 8. Примечания

- Всё работает **офлайн**; изображение **ориентировочное**, это не фотография и
  **не средство автоматической идентификации личности** — инструмент помощи
  оператору.
- Для `.exe` на другом компьютере нужен системный компонент **WebView2 Runtime**
  (в Windows 10/11 есть по умолчанию; иначе ставится бесплатным мини-установщиком
  от Microsoft).
- Стиль портрета — иллюстративный (так задумано на первом этапе).
