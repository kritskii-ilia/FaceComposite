# Карта книги «Типы и элементы внешности» (ТЭВ)

Снетков В.А., Зинин А.М., Виниченко И.Ф. — изд. ВНИИ МВД СССР, 1970.
Рисунки Н.А. Исакова и А.А. Сидякина. 140 страниц, **771 рисунок**.

Это рабочая карта PDF для извлечения деталей лица в проект. Книга задумана как
конструктор фоторобота: каждый элемент нарисован отдельно, портрет собирается
наложением выбранных рисунков черт друг на друга (методика прямо описана во
введении). Это совпадает с выбранным подходом — собирать лицо из деталей книги.

## Ключевые факты PDF

- Файл: 203 страницы, **скан без текстового слоя** (читается только как изображения).
- **Смещение нумерации: pdf = страница книги + 5.** (книжная стр. 73 = pdf 78.)
- **Контент книги: pdf 8 – 143** (книжные стр. 3 – 138).
- **Посторонние картинки (мусор): pdf ~146 – 203** — статуи, мугшоты, цитаты,
  ковид-билборды с водяным знаком PicCollage. При извлечении ОБЯЗАТЕЛЬНО отсечь.
- Рисунки — штриховые/точечные (перо), фас и часто профиль (вариант «а»),
  единый стиль. Именно тот «живой» вид, к которому стремимся.

## Как устроена опись типов (пример: Губы, книжн. стр. 115)

Иерархия с точными описаниями каждого рисунка, например:
- **I. Губы тонкие** — Рис. 29–34 (контур верхней каймы: извилистый/ломаный;
  углы рта: приподнятые/горизонтальные/опущенные).
- **II. Губы средние** — Рис. 35–59:
  - А. Верхняя губа шире нижней (Рис. 35–38)
  - Б. Верхняя губа уже нижней (Рис. 39–57)
  - В. Обе губы одинаковые (Рис. 58–59…)
- (далее — губы полные и т.д.), фас Рис. 71–82 и профильные варианты «а».

Такая же логика — у каждого элемента: несколько категорий, в каждой
пронумерованные рисунки с описанием различий. Это готовая таксономия и готовые
фразы для авто-подбора по описанию (наш NLU-слой).
